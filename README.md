# EvitaniaCalc
Item requirement calculator for the game Evitania Online

All recipes added (except for Hard act 2 ice mammoth) 
Separated item lists and drop downs per act
Added functionality for custom user entries being sorted by acts if desired (or creating new categories)

if you want to add more items, go the the "setup" tab and manually add the recipes - it will be saved locally on you computer.
optionally, export everything as a json file (the calculator can do that for you) and share it on discord or ping Yama at yamauti@gmail.com - he will then incorporate the new recipes
